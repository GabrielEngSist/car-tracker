# ClientApp entry (wrapper)

Este diretório só **reencamina** comandos npm para o SPA na pasta **pai**.

- Desenvolvimento: use `npm run dev` **a partir da raiz** `Car.Tracker.Presentation/` (recomendado), ou `npm run dev` daqui mesmo.
- Não há `package.json` de dependências aqui — `node_modules` fica apenas na raiz do frontend.
